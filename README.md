# Linkyboss

A conversational interview app that helps founders discover their voice profile for LinkedIn content.

## Setup

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com
3. Import the repo
4. Deploy (no config needed)
